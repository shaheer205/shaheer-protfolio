# Shaheer — Python Developer Portfolio

> Premium dark + glassmorphism portfolio for **Shaheer** — Python Developer · Automation Engineer · AI Enthusiast.

## ✨ Highlights
- Dark theme with **glassmorphism** & neon accents
- Hero **typing animation** + animated **particle field**
- Scroll-reveal animations, smooth scroll, sticky navbar with active section highlight
- Modular components, semantic HSL design tokens
- Fully responsive, SEO meta tags, basic a11y
- Back-to-top button, contact form with validation

## 🗂 Sections
Hero · About · Skills · Projects · Services · Timeline · Contact · Footer

## 🚀 Run locally
```bash
npm install
npm run dev
```

## 📦 Build
```bash
npm run build
npm run preview
```

## 🌐 Deploy
- **Netlify** — drag `dist/` to netlify.app/drop
- **Vercel** — `vercel --prod`
- **GitHub Pages** — push `dist/` to `gh-pages`

## 📁 Structure
```
src/
├── components/portfolio/   # All section components
├── hooks/useTypewriter.ts
├── pages/Index.tsx

└── main.tsx

index.html                  # SEO meta tags

```
